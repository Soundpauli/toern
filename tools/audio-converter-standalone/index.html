SD CARD

Download the default TŒRN SD card content — sample packs, settings, and folder layout. Extract the ZIP and copy the contents to your microSD card.

Download SD-CARD-CONTENT.zip

Recommended card

  • SanDisk Ultra microSD (grey label)
  • Speed class: UHS-I (Class 10)
  • Capacity: up to 64 GB
  • Use a genuine card from a trusted retailer — counterfeits often cause read errors or missing samples.

How to format

  1. Insert the microSD card into your computer with a card reader (not the TŒRN unit).
  2. Download and run the SD Memory Card Formatter from the SD Association.
  3. Select your card, leave Format type on Quick format, and click Format. Cards up to 32 GB are formatted as FAT32; 64 GB cards use exFAT — both work with TŒRN.
  4. Extract SD-CARD-CONTENT.zip and copy all files and folders to the root of the card (not inside a subfolder).
  5. Eject the card safely, insert it into TŒRN, and power on.

Avoid formatting through Windows Explorer or macOS Disk Utility if possible — the SD Association tool sets the correct partition layout for removable media.

SD card layout

├── settings.txt              # Device settings backup (menu, BPM, etc.)

│   ├── 2.wav                 # Voice 2
