
Basics:

* Patterns and Samplepacks are not connected in any way.
* A Pattern is just note triggers + filters, efx + parameters. 
* Used Samples and Soudpacks need to be saved as PACK, and PACKS need to be loaded individually - this enables more flexibility.
* Samplepacks and patterns both can be loaded at runtime individually without affecting settings like filters or bpm.

AutoLoad/Save:
	* Only on Pause, TOERN autosaves pattern / triggers / filters. 
	* SAMPLES and Soundpacks does NOT autosave (see. basics)
	* After turning the unit on, autosaved pattern + parameters + filters gets loaded.
	

Touch Buttons:
	Touch left: Toggle between FullMode(F) and SingleMode(S). TOERN always starts in FullMode.
	Touch right: Opens main menu
	Touch front: starts a direct recording on the current channel (line/mic) until released (Important: CAN NOT BE SAVED YET!)

Short Clicks:
	E1:Press: REMOVE a note
	E2:Press:toggle MUTE of channel
	E3:Press: Toggle PLAY / PAUSE
	E4:Press: SET a note

Long Clicks:
	E1: Long press+move for remove multiple notes
	E2: Long press Toggles SOLO for current Channel
	E3: Long Press opens FILTERS (+Parameter/Efx/Synth) Mode for the current cannel
	E4: Long press+move to set multiple notes (aka "draw")


FILTERS:
Different Pages for different Filters/EFX/etc.:
> Touch Right: cycles through all pages
> Touch Left: Exit
> E4: LongPress: Resets filters of that page to default

1: PFRB (fully implemented)
	-P: Pass(low+high): (Default/Bypass is Value=15 / middle)
	-F: FilterFreq for Pass
	-R: Reverb. Works on channels 1+2,7+8 and InstrumentVoice(11), and synthVoices(13+14)
	-B: Bitcrusher

3: ADSR: (fully implemented for SynthVoices 11+13+14, nearly no effect on 1-8)
	-A: "ATTC": Attack
	-D:	"DCAY": Decay
	-S; "SUST": Sustain
	-R: "RLSE": Release

4: CRFO (only usable for channel 11) 
	-C: "CUT": Cutoff
	-R: "RES": Resonance
	-F: "FLT": Filterfrequency
	-O: "OCTV": Octave (not yet implemented)

5: WICS (only usable for channel 11)
	-W: "WAVE": Waveform (not yet implemented)
	-I: "INST": Instrument:  Can be 1-8: BA(ss), KE(ys), CH(iptune), PA(d), WO(w), OR(gan), FL(ute), LE(ad), AR(p), BR(ass)
	-C: "CENT": Cent / Octave
	-S: "SEMI": Halbton

6: RDO[W] 
* (Channel 1-8[RDP] / 1-3[RDOS] / 13+14[RDOW])
	-R: "RES": Resonance (not yet implemented)
	-D: "DTNE": Detune
	-O: "OCT": Octave (not yet implemented)
	[-S: "SND": Sound] (not yet implemented)
	[-W: "WAVE": Waveform] (not yet implemented)

7: NDPT (Channel 1-3), 
* 3 Different Drums: BaseDrum(Channel 1), HiHat(Channel 2) and Snare (Channel 3), only usable if SND=DRUM is selected in RDOS
* not yet fully implemented
	-N: "NOTE": Note of Drum
	-D: "DCAY": Decay of Drum
	-P: "PITC": Pitch of Drum
	-T: "TYPE": Type of Drum


Main Menu:
1:LOAD/SAVE PATTERN ("file")
	E1:Press loads
	E2:Press saves
	E3:turn: Selects if pattern should be loaded with all its FILTERS("+") or without
	E4:turn: Selects Pattern slot
	Touch Right or Left: EXIT

	* loading a NEW file (blue numbers) auto resets all Filters, as well as BPM, as long the option ("+") isnt selected with E3.e
	* loading an exisiting file, also loads all saved Filters
	* saving a file also saves all filters
	* saving a file does NOT save samples (yet), they have to be loaded / saved from SAMPLEPACK menu.

	

2:LOAD/SAVE SAMPLEPACK ("pack")
	E1:Press loads
	E2:Press saves
	E4:turn: Selects PACK slot
	Touch Right or Left: EXIT

	* Saves the current used Samples (which where loaded from SD) to an own samplepack which can be re-used later in the same or another pattern.
	* Important: Edits to Samplelength as well as "live-looping"-recorded samples from using "Toch front" are NOT saved to samplepacks (yet)

3: Record-Input Select
	E4: Press: Toggles beween MIC/LINE
	E3: Turn: Sets the Mic-In-Level (doesnt work well, sadly)

4: LOAD WAVE/SAMPLE ("wave") 
	E1:Press loads SAMPLE to channel (see color)
	E1:Turn: Selects SampleStart position
	E2:Press+hold: record MIC/LINE to SAMPLESLOT (important: overwrites without asking!)
	E4:turn: Selects SAMPLE FOLDER (1-9)
	E4:turn: Selects SAMPLE slot (0-99)
	Touch Right or Left: EXIT

	*LOAD/SAVE only works if a sampleslot is selected with the ypos of the cursor (1-8), otherwise it shows "(-)"

5: SET BPM+VOLUME+BRIGHTNESS ("BPM")
	E2:Turn: Selects brightness
	E3:turn: Selects volume
	E4:turn: Selects bpm
	Touch Right or Left: EXIT

6: Midi Clock ("clck")
	E4: Press: Toggles Intern(TOERN) / Extern (via MIDI-TRS)
	Touch Right or Left: EXIT

7: Midi Channel ("midi")
	E4: Press: Toggles YPOS(cursorpos/channelColor) / MIDI (omnichannel)
	Touch Right or Left: EXIT

8:Transport ("trsp")
	E4: Press: Toggles ON / OFF for TOERN sending/receiving MIDI-play/pause signals
	Touch Right or Left: EXIT

9: PATTERN_MODE ("PMOD", >Matzemode")
	E4: Press: 	Toggles ON/OFF
	Touch Right or Left: EXIT

	* Patternmode ON just repeats the currently visible/active page in an pattern ;)

10: FLOW_MODE
	E4: Press: Toggles ON / OFF. 
	Touch Right or Left: EXIT

	* Flowmode ON always shows the active page while playing ("What you see is what you get")

11: Recording Button ("rec")
	E4: Press: Toggles between OFF / SENS / -CON / +CON
	Touch Right or Left: EXIT

	* SENS uses the touch sensor in front panel (defaultsetting)
	* -CON/+CON uses a connected footswitch ()

12: Clear ("clr") notes when recording
	E4: Press: Toggles between FIX / OFF / ON
	Touch Right or Left: EXIT

	* FIX uses the already set note triggers for recording (default setting)
	* ON does clear the patterns/page note triggers and sets a new trigger at the time/moment of recording
	* off doesnt add or delete any note triggers when record is touched/pressed

13: Preview-Volume
	E4: Press: Toggles LOW/MID/HIGH/SPLT
	Touch Right or Left: EXIT

	* how loud the preview of SAMPLE-Selection should play.
	* SPLT only uses the LEFT channel of the Audio, leaving ALL other SOUNDS of TOERN play on the RIGHT channel! 
	* SPLT enables to use "silent" preview routing in livesets

14: Live-Loop-Monitor-Volume
	E4: Press: Toggles OFF/LOW/MED/HIGH/FULL
	Touch Right or Left: EXIT

	* How loud the Recorded (MIC/LINE) INPUT should be routed to monitor.

15: Reset
	E4: Press: Resets all FILTERS (+Parameters/+FX,etc) of the Pattern do default. 
	Touch Right or Left: EXIT

	* Helps when sound is missing or distorted after a lot of action

16: Set (not ready yet)
	Touch Right or Left: EXIT
