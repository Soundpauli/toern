The page row

Turn encoder 1 until the cursor sits on the top strip of the matrix — that is the page row (row 16, y==16). It shows your 16 pattern pages as a row of dots, and it is the only place the page-wide tools fire.

Cursor lock: every page-row edit below — delete page, copy/paste, clear page, random fill, note-shift — runs only when the cursor is on row 16. On a voice row (y ≠ 16) the same presses paint, erase, mute, or do nothing. No exception.
You are on the page row when the hint strip on row 1 changes colour and the matching encoder rings light up. That colour change is the cue that power tools are armed — if the hints look like normal draw/single, you are still on a voice row.

What lives here (cursor on row 16 required):

  • Delete a page — unpaint on the page strip (clears that page’s notes)
  • Copy & paste a page (draw = all voices; single = focused voice only)
  • Clear page for the focused voice — single mode, encoder 1 short (current edit page)
  • Random fill for the focused voice — single mode only
  • Note-shift — slide the focused voice in time / across rows — single mode only

Page strip colours

Each LED on row 16 is one pattern page. Idle pages use hasNotes so you can see which slots still hold triggers:

Delete a page — unpaint on the strip

On the page row, unpaint does not erase a single step. It clears the whole page under the cursor — how you “delete” a slot that still shows as has-notes (dark blue).

  1. Move the cursor onto the page row (row 16) so the hint strip lights up.
  2. Turn encoder 4 until the cursor sits on the page LED you want gone (a dark-blue “has notes” slot).
  3. Unpaint that cell with encoder 1 in DRAW = L+R (the erase hand) — press enc 1 on that strip LED.

What gets wiped:

  • Draw mode — every voice on that page (all triggers for those 16 steps).
  • Single mode — only the focused voice on that page (other lanes keep their notes; the page LED stays “has notes” until every voice is empty).

When the page is fully empty, its strip LED drops to very dark red. Playback range (lastPage) updates so trailing empty pages fall out of the walk.

R mode: encoder 4 on the page strip is copy / paste, not erase. Flip to L+R (or use the clear-page gestures below) when you want to wipe a page from the strip.
No undo. Unpainting a page strip slot is immediate. Save to FILE first if you might want that page back.
Two different “clears”: unpaint on the strip (cursor y==16) targets the page LED under the cursor. Encoder 1 short in single mode on the page row clears the current edit page for the focused voice only — handy mid-edit, different target. The enc 1 + enc 4 long chord clears the current edit page too and works from any row (all voices in draw; focused voice in single).

Copy armed

After the first encoder 4 tap, the page strip goes dark and the hints jump to large blue (enc 1) and large yellow (enc 4). Full copy / paste flow: Advanced Sequencing → Copy & paste.

Single mode on the page row

Hints: red (enc 1), white (enc 2), yellow (enc 4). Same copy/paste idea, but it only grabs the focused voice — so you can copy a hi-hat lane and paste it onto another voice after you change focus.
